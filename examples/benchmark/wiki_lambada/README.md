## 运行预测脚本

验证Lambada数据集，运行以下脚本：
```
python eval.py \
--model_name_or_path /path/to/your/model \
--batch_size 4 \
--eval_path /path/to/your/dataset/lambada_test.jsonl \
--tensor_parallel_degree 1 \
--cloze_eval
```

验证WikiText数据集，运行以下脚本：
```
python eval.py \
--model_name_or_path /path/to/your/model \
--batch_size 4 \
--eval_path /path/to/your/dataset/wikitext-103/wiki.valid.tokens \
--tensor_parallel_degree 1
```
