# ChatPaper
## 1. 功能概述
ChatPaper 【单篇/多篇】翻译、精读、文章chat问答，支持英译中。
## 2. 实现方式
```
pip install -r requirements.txt
pip install --upgrade paddle-pipelines
python chat_paper.py
```
