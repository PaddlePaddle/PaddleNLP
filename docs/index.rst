欢迎使用PaddleNLP
==================
`PaddleNLP <https://github.com/PaddlePaddle/PaddleNLP>`_ 是基于飞桨框架的NLP领域最佳实践，
集成了丰富的数据集、网络结构、预训练模型、评价指标、数据处理等通用功能。具备......等特性。
