<h2><a id="user-content-预训练模型使用方法" class="anchor" aria-hidden="true" href="#预训练模型使用方法"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>预训练模型使用方法</h2>
<p>PaddleNLP Transformer API在提丰富预训练模型的同时，也降低了用户的使用门槛。只需十几行代码，用户即可完成模型加载和下游任务Fine-tuning。</p>
<div class="highlight highlight-source-python position-relative" data-snippet-clipboard-copy-content="from functools import partial
import numpy as np

import paddle
from paddlenlp.datasets import load_dataset
from paddlenlp.transformers import BertForSequenceClassification, BertTokenizer

train_ds, dev_ds, test_ds = load_dataset(&quot;chnsenticorp&quot;, splits=[&quot;train&quot;, &quot;dev&quot;, &quot;test&quot;])

model = BertForSequenceClassification.from_pretrained(&quot;bert-wwm-chinese&quot;, num_classes=len(train_ds.label_list))

tokenizer = BertTokenizer.from_pretrained(&quot;bert-wwm-chinese&quot;)

def convert_example(example, tokenizer):
    encoded_inputs = tokenizer(text=example[&quot;text&quot;], max_seq_len=512, pad_to_max_seq_len=True)
    return tuple([np.array(x, dtype=&quot;int64&quot;) for x in [
            encoded_inputs[&quot;input_ids&quot;], encoded_inputs[&quot;token_type_ids&quot;], [example[&quot;label&quot;]]]])
train_ds = train_ds.map(partial(convert_example, tokenizer=tokenizer))

batch_sampler = paddle.io.BatchSampler(dataset=train_ds, batch_size=8, shuffle=True)
train_data_loader = paddle.io.DataLoader(dataset=train_ds, batch_sampler=batch_sampler, return_list=True)

optimizer = paddle.optimizer.AdamW(learning_rate=0.001, parameters=model.parameters())

criterion = paddle.nn.loss.CrossEntropyLoss()

for input_ids, token_type_ids, labels in train_data_loader():
    logits = model(input_ids, token_type_ids)
    loss = criterion(logits, labels)
    probs = paddle.nn.functional.softmax(logits, axis=1)
    loss.backward()
    optimizer.step()
    optimizer.clear_grad()

<ol>
<li>加载数据集：PaddleNLP内置了多种数据集，用户可以一键导入所需的数据集。</li>
<li>加载预训练模型：PaddleNLP的预训练模型可以很容易地通过<code>from_pretrained()</code>方法加载。第一个参数是汇总表中对应的 <code>Pretrained Weight</code>，可加载对应的预训练权重。<code>BertForSequenceClassification</code>初始化<code>__init__</code>所需的其他参数，如<code>num_classes</code>等，也是通过<code>from_pretrained()</code>传入。<code>Tokenizer</code>使用同样的<code>from_pretrained</code>方法加载。</li>
<li>通过Dataset的map函数，使用tokenizer将dataset从原始文本处理成模型的输入。</li>
<li>定义BatchSampler和DataLoader，shuffle数据、组合Batch。</li>
<li>定义训练所需的优化器，loss函数等，就可以开始进行模型fine-tune任务。</li>
</ol>
