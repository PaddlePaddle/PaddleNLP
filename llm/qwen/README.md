# Qwen

## 1.模型介绍

[通义千问-7B（Qwen-7B）](https://arxiv.org/abs/2205.01068) 是阿里云研发的通义千问大模型系列的70亿参数规模的模型。Qwen-7B是基于Transformer的大语言模型, 在超大规模的预训练数据上进行训练得到。预训练数据类型多样，覆盖广泛，包括大量网络文本、专业书籍、代码等。

**支持模型权重:**
| Model             |
|-------------------|
| qwen/qwen-7b      |
| qwen/qwen-7b-chat |
