# 飞桨大语言模型
飞桨大模型套件PaddleFleetX是基于PaddlePaddle的4D分布式并行能力的大模型全流程套件，旨在提供高性能、灵活易用大模型工具，可以根据自己的需求轻易来定制化百亿和千亿大模型训练，同时支持高性能的压缩推理和服务化，最终使用大模型能力提升业务效果。
